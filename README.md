# needed

## 프로젝트 요약

> 👉 I need YOU for the world’s leading companies!

1. 다양한 일자리를 카테고리별로 나눠 구직자에게 접근성이 용이하게 합니다.
2. 잡포스팅 외에도 다양한 이벤트, 직군별 연봉정보 등을 제공합니다.
3. 각종 테마에 대하여 서로 의견을 주고받을 수 있는 커리어 커뮤니티를 가지고 있습니다.
